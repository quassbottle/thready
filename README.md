## Thready

Simple **anonymous** imageboard (without images for now xd) backend application made with Nest

## Stack

- Node.js
- TypeScript
- Nest.js
- Prisma ORM

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
